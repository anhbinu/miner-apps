# Mining Tools

## Mining Eth coin

Steps:

1. Open file ```win-claymore-eth.bat```
2. Change pools and wallet ID to yours
3. then RUN the file

## Mine other Coin

1. ETH: ```win-claymore-eth.bat```
2. ETC: ```win-claymore-etc.bat```
3. ZCash: ```win-claymore-zcash.bat```
